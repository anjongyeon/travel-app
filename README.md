### 설치 과정
1. 저장소 클론
```bash
git clone https://github.com/your-username/my-first-app-main
cd journey-flow
```

2. 의존성 패키지 설치
```bash
npm install
# 또는
yarn install
```

3. 개발 서버 실행
```bash
npm start
# 또는
yarn start
```
- 브라우저에서 http://localhost:3000 으로 접속하여 애플리케이션 사용
